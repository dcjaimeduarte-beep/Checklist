# /review

Revisão de código em linguagem simples.

## O que fazer

1. Atuar como o subagente definido em **`.cursor/agents/code-review.md`**: conhecer o contexto do monorepo via **`AGENTS.md`** e convenções em **`.cursor/rules/`**.

2. Se o utilizador não indicou ficheiros ou diff, pedir **uma** frase sobre o que rever ou usar o contexto atual / alterações recentes.

3. Resposta **objetiva**, sem “textão”, em **três blocos** quando fizer sentido:
   - **Bloquear / corrigir já**
   - **Melhorar**
   - **Opcional**

4. Linguagem acessível a qualquer pessoa da equipa; exemplos concretos só quando ajudarem.
